2. Cancel Booking

Users must cancel and free seats.

Backend flow

Cancel Booking
→ update booking status CANCELLED
→ update payment status REFUND_PENDING
→ release seats


API:

PUT /booking/cancel/{bookingId}


Database update:

booking.status = CANCELLED
